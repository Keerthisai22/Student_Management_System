# Student Management System (Python)

A menu-driven Python application to manage student records using a dictionary.  
The system allows adding, updating, viewing, and deleting student details.

## Features
- Add new student details (roll number, name, marks)
- Update student marks
- View all students
- Delete student records
- Menu-driven interface
- Runs continuously until exit

## Technologies Used
- Python 3

## How to Run
1. Download or clone the repository
2. Run the program:
   python student_management_system.py
